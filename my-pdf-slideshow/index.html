[MyPdfSlideShow - a device local PDF viewer] - press "." to toggle on/off this menu
    <--Please choose a PDF file on your device.     Page: /

Usage - key bindings
Key Command
SPACE or f go to next page
b go to previous page
p go to first page
g go to last page
press digits + ENTER go to specified page
r reload page
o open local folder
. toggle show/hide this menu
Usage - mouse control
Operation Behavior
click a link in the page open a url in a new tab (named 'mypdfslideshow')
click a link in the page with ctrl-key pressed open a url in a new tab (named '_blank')
